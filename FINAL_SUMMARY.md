## 🚀 如何使用

### 快速開始
```bash
# 1. 運行遊戲
python main.py

# 2. 查看演示
python demo.py

# 3. 運行測試
python tests/test_stories.py
```

### 開發設置
```bash
# 1. 創建虛擬環境
python -m venv venv
venv\Scripts\activate  # Windows

# 2. 安裝開發工具 (可選)
pip install pytest black flake8 mypy

# 3. 開始開發
# 編輯 stories/halloween.json 添加新場景
# 運行測試驗證
```
